# sCalendar

A simple, clean weekly calendar app for iOS built with SwiftUI.

## Features

- **Weekly View**: Browse your schedule by week with an intuitive interface
- **Calendar Integration**: Seamlessly connects with your existing iOS calendars via EventKit
- **Event Management**: Create, view, and edit calendar events
- **Search**: Quickly find events by title or location
- **Calendar Customization**: Show/hide calendars and customize colors
- **iCloud Sync**: Preferences sync across your devices via iCloud
- **Localization**: Multi-language support

## Requirements

- iOS 16.0+
- Xcode 15.0+
- Swift 5.9+

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/thomasriehn/sCalendar.git
   ```

2. Open `sCalendar.xcodeproj` in Xcode

3. Build and run on your device or simulator

## Privacy

sCalendar respects your privacy. All calendar data stays on your device and is managed by Apple's EventKit. The app does not collect analytics, track usage, or share data with third parties. See [PRIVACY_POLICY.md](PRIVACY_POLICY.md) for details.
